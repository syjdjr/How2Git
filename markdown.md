## 1. 제목 (Heading)
- level 1부터 level 6까지 6가지의 크기로 표현할 수 있다. 
- level이 높아질수록 제목의 크기가 작아진다. 

> #### [ 표현 방법 1 ] : # 사용하기 
>>  사용법 : #를 제목 **앞**에 원하는 **level의 수만큼** 붙여준다. 
>> 표현단계: level1 ~ level6 까지 **모든 단계**를 나타낼 수 있다.  
>
> #### [ 표현 방법 2 ] : = 사용하기
>>사용법 : =를 제목 **아래**에 **아무 수만큼** 붙여준다. 
>>표현 단계 : **level 1만** 나타낸다. 
>
> #### [표현 방법 3] : - 사용하기
>> 사용법 : -를 제목 **아래**에 아무 수만큼 붙여준다. 
>>표현 단계 : **level 2만** 나타낸다.
>
> #### [표현 방법 4] : HTML tag 사용하기
>> 사용법 : <> 안에 h와 표현할 level의 숫자를 넣어준다. 그 뒤 heading 내용을 써넣은뒤 space 없이 <>를 붙여준다. 이때, <> 안에 /와 h 그리고 표현할 level의 숫자를 넣어준다. *예시는 지침서의 14번 항목 HTML 내용을 참조한다.*

## 5. 강조 (emphasis) 

## Bold 
- 강조 표현 중 하나로 글씨를 진하게 하여 강조한다. 

> #### [표현 방법 1] : ** 사용하기 
>> 사용법 : 강조하고 싶은 **단어의 앞과 뒤**에 각각 **을 붙여준다. 
>> 특징 : 맨 앞, 맨 끝, 가운데 단어까지 위치 상관 없이 **모두** Bold 표현 가능하다.
>> *주의 : \*\*과 단어 사이에 space를 두지 않아야 한다.*
>
> #### [표현 방법 2] : __ 사용하기 
>> 사용법 : 강조하고 싶은 **단어의 앞과 뒤**에 각각 \_\_를 붙여준다.  
>> 특징 : 맨 앞과 맨 끝 단어 표현 가능하지만, **가운데 단어는 표현이 불가하다.** 
>> *주의 : \_\_과 단어 사이에 space를 두지 않아야 한다.*
>
> #### [표현 방법 3] : HTML tag 사용하기 
>> 사용법 : 강조할 단어나 문장의 앞에 \<strong\>을 붙여 강조가 시작됨을 표시하고, 강조할 단어나 문장의 끝에 \<\/strong\>을 붙여 강조가 끝남을 표시한다.

## Italic 
- 강조 표현 중 하나로 글씨를 Italic체로, 기본 글씨체와 다르게 표현하여 강조 시킨다. 

> #### [표현 방법 1] : * 사용하기 
>> 사용법 : 강조하고 싶은 **단어나 문장의 앞과 뒤**에 각각 *를 붙여준다.
>> 특징 : 맨 앞, 맨끝, 가운데 단어까지 위치 상관 없이 **모두** 표현 가능하다.
>> *주의 : \*와 단어 사이에 space를 두지 않아야 한다.* 
>
> #### [표현 방법 2] : _ 사용하기 
>> 사용법 : 강조하고 싶은 **단어나 문장의 앞과 뒤**에 각각 _ 를 붙여준다.
>> 특징 : 맨 앞과 맨 끝 단어 표현 가능하지만, **가운데 단어는 표현이 불가하다.** 
>> *주의 : \_ 와 단어 사이에 space를 두지 않아야 한다.* 
>
> #### [표현 방법 3] : HTML tag 사용하기
>> 사용법 : 강조할 단어나 문장의 앞에 \<em\>을 붙여 강조가 시작됨을 표시하고, 강조할 단어나 문장의 끝에 \<\/em\>을 붙여 강조가 끝남을 표시한다.

### Bold 와 Italic 한번에 표현하기
> #### [표현 방법 1] : *** 사용하기
>> 사용법 : 강조하고 싶은 **단어나 문장의 앞과 뒤**에 각각 ***를 붙여준다. 
>> 특징 : 맨 앞, 맨끝, 가운데 단어까지 위치 상관 없이 **모두** 표현 가능하다.
>> *주의 : \*\*\*와 단어 사이에 space를 두지 않아야 한다.* 
>
> #### [표현 방법 2] : ___ 사용하기 
>> 사용법 : 강조하고 싶은 **단어나 문장의 앞과 뒤**에 **space 없이**각각 \_\_\_를 붙여준다.
>> 특징 : 맨 앞과 맨 끝 단어 표현 가능하지만, **가운데 단어는 표현이 불가하다.** 
>> *주의 : \_\_\_와 단어 사이에 space를 두지 않아야 한다.* 
>
> #### [표현 방법 3] : _와 *를 함께 사용하기 
>> 사용법 : 강조하고 싶은 **단어나 문장의 앞과 뒤**에 각각 __* 혹은 **_ 를 **대칭적**으로 배치한다. 
>> *주의 : __\* 혹은 \*\*\_ 와 단어 사이에 space를 두지 않아야 한다.*
> 
> #### [표현 방법 4] : HTML tag 사용하기 
>> 사용법 : 강조하고 싶은 단어나 문장의 앞에 \<strong\>\<em\>을 붙여 강조가 시작됨을 알리고, 강조할 단어나 문장의 끝에 \<\/em\>\<\/strong\>을 붙여 강조가 끝남을 알린다.

## 6. List

### 순서 있는 Lists
- 들여쓰기 하여 새로운 순서 Lists를 만들 수 있다. 
 
> #### [표현 방법 1] : 숫자 순서대로 표기 
>> 사용법 : **숫자.** 의 형식을 사용하되 이때 숫자는 **숫자 1부터 순서**대로 사용한다. 
>> *주의 : \. 이 아닌 다른 기호 사용 불가하다*
>
> #### [표현 방법 2] : 한가지 숫자로 표기 
>> 사용법 : **숫자.** 의 형식을 사용하되 이때 숫자는 **숫자 1만** 사용한다. 
>> *주의 : \. 이 아닌 다른 기호 사용 불가하다.* 
>
> #### [표현 방법 3] : 무작위 숫자로 표기 
>> 사용법 : **숫자.** 의 형식을 사용하되 이때 숫자는 **숫자 1부터 무작위 순서**대로 사용한다. 
>> *주의 : . *(point)* 이 아닌 다른 기호 사용 불가하다.*
> #### [표현 방법 4] : HTML tag 사용하기 
>> 사용법 : \<ol\>를 사용하여 순서가 있는 리스트가 시작됨을 알리고, 다음 line에 \<li\>리스트 내용\<\/li> 형식을 지켜 list의 순서를 부여한다. 마지막 순서의 list의 다음 line에 \<\/ol>을 사용하여 리스트가 끝남을 알린다.

>> 사용법 : \<a href="URL"\>텍스트\<\/a\> 의 형식에 맞추어 사용한다. 
>> ##### [가독성을 위한 옵션]
>> **[링크 title 만들기]**: 링크를 넣은 **() 에서 링크 뒤**에 title을 넣는다. 
>> **[빠르게 URL과 메일 주소를 나타내는 방법]** : **< > 괄호 안**에 URL과 메일 주소를 넣는다

### [ reference-style links] 
- Markdown에서 URL이 한눈에 보기 쉽게 만들어주는 특별한 링크 스타일이다.
- 길고 복잡한 본래 링크 대신 간소화 해주어 문단에서 더 잘 표현할 수 있도록 해준다. 

> #### [표현 방법] : 
>> **[1단계] : \[ \]\[ \] 이용하기**
>> 1. 첫번째 \[ \]안에 **link를 표현**할 단어나 문장을 넣는다.
>> 2. 두번째 \[ \] 안에 link로 자동 이동할 수 있는 **label**을 넣는다. 
*이때, label은 문자, 숫자, 여백, 구두점으로 표현 가능하다.* 
>> *주의 : 첫번째 \[\] 와 두번째 \[\] 사이에 space가 있으면 안된다.*  
>
>> #### [2단계] : label에 링크 저장하기 
>> 1. [ ] 안에 label을 넣는다. 
>> 2. [ ] 뒤에 :을 붙인다
>> 3. : [ ] 뒤에 **적어도 한번의 space**를 둔 후 링크의 URL을 붙인다. 
    >> - 이때 **< >안에 URL**을 넣거나 **URL만**으로도 표현 가능하다.
>> 4. 링크의 title을 아래의 방법을 이용하여 자유롭게 넣을 수도 있다. 
    >> - [방법1] : "" (double quotes) 사용하기
    >> - [방법2] : '' (single quotes) 사용하기
    >> - [방법3] : () (parentheses) 사용하기 
>> *주의 : URL을 표현할 때, spaces는 사용 불가 이기 때문에 **%20**을 사용하여 spaces를 없앤다.*

## 10. Image 
> ### [image 넣는 방법] 
>> 1. line 제일 **앞**에 !를 사용한다. 
>> 2. 그 뒤에 **space없이** [ ] 를 사용하는데, 이때 **path** 혹은 **URL**를 [ ] 안에 넣어준다. 
>> 3. URL의 title을 붙일 수도 있다.  
> ### [ Linking images]
>- 이미지의 링크를 걸어 이미지를 추가하는 것이다. 
>
> [Linking image 만드는 방법] : 
>> 1. [ ] 안에 ![ ] 을 넣어준다. 이때 ![ ]안에 **path** 혹은 **URL**을 **space 없이** 넣어준다. 
>> 2. 그 뒤 ( ) 안에 링크를 삽입한다. 

|방법|사용 형식 *예시*|
|---|---|
|image|![텍스트](이미지의 경로)<br> ![텍스트](이미지의 URL)|
|linking image|[![텍스트](이미지의 경로)](링크 URL)<br> [![텍스트](이미지의 URL)](링크 URL)|

## 12. HTML
- Markdown에서 HTML tag를 사용할 수 있다.
- HTML tags을 이용하여 문자의 컬러나 이미지의 크기 조정 등 다양한 요소를 쉽게 활용할 수 있다. 
- 하지만, Markdown이 HTML을 전부 지원하는 것은 아니다. 

| Syntax |사용 형식 |
|-----|-------|
|Heading| \<h숫자\>내용\<\/h숫자\>|
|paragraphs| \<p\>내용\<\/p\>|
|line breaks| \<p\>내용\<br\> \<\/p\>|
|bold|\<strong\>강조할 단어나 구절\<\/strong\>|
|Italic|\<em\>강조할 단어나 구절\<\/em\>|
|bold와 Italic 한번에 표시| \<strong\>\<em\>강조할 단어나 구절\<\/em\>\<\/strong\>|
|ordered list| \<ol\> <br> \<li\>내용\<\/li\> <br> \<\/ol\>|
|unordered list| \<ul\> <br> \<li\>내용\<\/li\> <br> \<\/ul\>|
|code|\<code\>단어나 구절\<\/code\>|
|link|\<a href="URL"\>텍스트\<\/a\>|
